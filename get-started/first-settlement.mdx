This walks the first run end-to-end and shows what a single **Generate** actually does — so the preview stops feeling like magic and starts feeling like a tool you direct.

## Set the stage

<Steps>
  <Step title="Place the Generator and select it">
    **Window → OmniScape → Place OmniScape Generator.** The Control Center binds to the selected actor (you can switch actors from the title-bar selector when you have several).
  </Step>
  <Step title="Assign a Large building">
    On the **Build** tab, add one **Large** asset. A static mesh instances fast; a Blueprint runs full actor logic. This is the only required asset — it becomes the landmark at each node.
  </Step>
  <Step title="Choose where it lives">
    On the **Layout** tab, pick a **pattern** and set **Settlement size** and **Num Nodes**. Or load a [preset](/guides/presets) to start from a curated archetype, then swap in your art.
  </Step>
  <Step title="Generate">
    Click **Generate**. The 2D preview pauses briefly, then resolves the full layout.
  </Step>
</Steps>

## What one Generate pass does

A single run executes in order — knowing the order makes results predictable.

<AccordionGroup>
  <Accordion title="1 · Landmark nodes" icon="circle-dot">
    The pattern places landmark nodes, honouring any **manual anchors** you've dropped.
  </Accordion>
  <Accordion title="2 · Roads" icon="route">
    A pattern-specific network is laid, then junctions are cleaned up. Optional **intersection meshes** drop at hubs.
  </Accordion>
  <Accordion title="3 · Node tweaks" icon="move">
    Large buildings are nudged off road centrelines so landmarks sit cleanly.
  </Accordion>
  <Accordion title="4 · Buildings" icon="building">
    One **Large** per node, then **Medium** and **Small** (Cluster or Roadside), then optional **sprawl** controlled by **Edge Density**.
  </Accordion>
  <Accordion title="5 · Environment & ambient" icon="trees">
    Vegetation, rocks, debris, and — in Beta — fauna and audio zones where distance, zone, and slope rules allow.
  </Accordion>
  <Accordion title="6 · Walls, landscape & PCG" icon="shield">
    Walls generate if enabled; PCG export data refreshes; height-snap runs automatically. The **landscape paint bake** is a separate editor action so you can rerun or undo it.
  </Accordion>
</AccordionGroup>

## Lock it in

<Steps>
  <Step title="Iterate with the preview">
    Tune settings and watch the live map. Prefer **Generate** for heavy maps; **Auto-Generate** respawns full geometry on every edit and is much heavier.
  </Step>
  <Step title="Save to Level">
    When the composition is right, **Save to Level**. This is what makes the settlement survive reload and packaging.
  </Step>
</Steps>

<Warning>
  **Preview ≠ Auto-Generate.** The preview is a lightweight map; Auto rebuilds real geometry each change. Tune with preview, then Generate when ready.
</Warning>
