Most issues come down to save flow, collision, or a single setting. Start here.

## Quick diagnosis

| Symptom | Likely cause | Fix |
|---|---|---|
| Nothing spawns | No **Large** assets, or traces miss the ground | Add a Large asset on [Build](/guides/build); check [terrain & collision](#terrain--collision) |
| Gone after reload | Level not saved | [Save to Level](/get-started/save-to-level) after generating |
| Float or sink | Collision mismatch | [Terrain & collision](#terrain--collision) |
| Fall through roads/walls | Very old plugin build | Update from [Fab](https://fab.com/s/4aeff49faf04) |
| Empty preset list | Partial install | [Reinstall from Fab](/guides/presets) |
| Layout changed after update | Road stack vs Legacy | See [FAQ — installation](/help/faq) |
| Anchor M/S ignored | Roadside placement mode | Per-anchor lists are Cluster-only |
| Odd PIE preview | Showing the last bake | The preview pauses during a full generate |

## Terrain & collision

Use this when **nothing spawns**, actors **float or sink**, or roads feel wrong physically.

<AccordionGroup>
  <Accordion title="Ground must have CPU collision" icon="layers">
    Placement uses downward traces against walkable surfaces — **World Static** first, then a **Visibility** fallback. GPU-only or invisible-mismatch ground causes floats, sinks, and slope rejections. Make sure your terrain has CPU collision aligned with the surface.
  </Accordion>
  <Accordion title="Generated meshes use block-all collision" icon="shield">
    Roads (including intersections), walls, pillars, and gates use block-all collision so gameplay interacts with them. Need something different? Use your own meshes or add wrapper collision.
  </Accordion>
  <Accordion title="Third-party terrain" icon="mountain">
    Needs CPU collision aligned with the walkable surface. If everything oddly avoids the road, tune road **width** against **overlap padding**.
  </Accordion>
</AccordionGroup>

## Few or no medium/small buildings

Check tier **counts**, **Cluster vs Roadside**, **density**, and overlap or road clearance. Remember Roadside uses the global medium/small lists, not per-anchor overrides.

## Performance while tuning

Editor hitches usually mean the map is heavy. Reduce **radius**, **node count**, or **flora samples**, and disable **Auto-Generate** while you tune — the live preview updates without it.
