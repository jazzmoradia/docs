## Before a frustrated review

Before leaving a frustrated Fab review, try a quick comment on the support thread. Most issues turn out to be save flow, collision, or a single setting — a two-minute exchange usually sorts it, and it helps everyone who reads later.

<Tip>
  When you post, include your **engine version**, **what you clicked**, and a **screenshot** if you can. It's the fastest path to a precise answer.
</Tip>

## Docs assistant

A **NotebookLM docs assistant** trained on the OmniScape docs and codebase gives instant Q&A on setup, presets, and troubleshooting. Open it from the **developer note** (the note icon in the Control Center title bar), or the first-open prompt via **Window → OmniScape**.
