OmniScape is a partnership: you bring the art and the intent, OmniScape brings the spatial composition.

## You bring, OmniScape builds

<Columns cols={2}>
  <Card title="You add" icon="box">
    Meshes or Blueprints for **large / medium / small** buildings, and optionally roads, walls, and environment kits. Your scale, your style.
  </Card>
  <Card title="OmniScape adds" icon="building-2">
    Layout, zones, roads, walls, terrain placement, PCG export data, terrain paint bake, ambient flora and audio, and instanced environment — from one Generator.
  </Card>
</Columns>

<Note>

## Core feature set

| Feature | What it does |
|---|---|
| **Control Center** | One dockable panel — all controls, presets, 2D preview, generate, clear, and seed. |
| **Layout patterns** | Radial, Voronoi, Perlin, Axial, and Organic — each with direct per-pattern tuning. |
| **Placement modes** | Cluster (rings around nodes) or Roadside (buildings face and plot along roads). |
| **2D live preview** | The settlement map updates as you tune — roads, blocks, and footprints before you generate. |
| **Viewport preview** | Read streets and footprint hints directly in the level viewport before a full pass. |
| **Wall system** | Procedural perimeter walls — shape, coverage, gates, pillars, separate asset lists. |
| **Flora layers** | Vegetation, rock, and debris scatter; optional Foliage Type slot for harvest workflows. |
| **Anchors & volumes** | Fix layout nodes by hand; define inclusion / exclusion regions with spline volumes. |
| **Blueprint events** | `OnSettlementGenerated` / `OnSettlementCleared`, hookable from Blueprint or C++. |
| **Deterministic seeds** | Same seed + settings = same result; toggling one layer never reshuffles the others. |

### New in v3.0

| Feature | What it does |
|---|---|
| **Atlas** <span className="os-tag os-phase-layout">v3.0</span> | Import any real place — by name, `.osm`/`.geojson` file, or screenshot — as your layout skeleton. |
| **12 preset archetypes** <span className="os-tag os-phase-layout">v3.0</span> | Citadels, planned cities, hamlets, coastal villages, ruins, sacred precincts, and more. |
| **Semantic zones** <span className="os-tag os-phase-layout">v3.0</span> | District types (residential, market, sacred, civic) inherited by buildings, flora, fauna, and PCG. |
| **Road hierarchy** <span className="os-tag os-phase-structure">v3.0</span> | Arterial / Collector / Local classes with independent width, mesh, and terrain mode. |
| **Smarter walls** <span className="os-tag os-phase-structure">v3.0</span> | Concave-hull perimeters that hug irregular sites; evenly spaced gates; corner towers. |
| **Beta pillars** <span className="os-tag os-phase-ambient">v3.0</span> | Fauna, Ambient Audio, Landscape Bake, PCG Hand-off, and road landscape deform. |

## Ship games with it

Generate in the editor → **Save to Level** → cook. Players never need the Control Center — they see saved meshes and instanced components like any other placed content.
